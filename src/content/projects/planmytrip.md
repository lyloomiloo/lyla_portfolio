## The Problem

Trip planning tools are either ugly spreadsheets or rigid templates. I wanted something useful for planning — editable, data-connected — that also looks like something you'd want to look at.

## What It Does

A 15-day Europe Alps Tour planned and displayed in an editorial layout inspired by travel magazines. Click-to-edit everything. Live weather per destination. Built-in image search. Asymmetric photo galleries.

## Design Approach

The layout reproduces a Keynote presentation I designed — city intro slides with split typography, day pages with photo galleries and timelines, weather widgets. A planning tool that's also beautiful enough to screenshot and share.

## What I Learned

My first real frontend project built with Claude Code — I designed the visual system, wrote the data architecture and specification, then used AI-assisted development to build it. The process itself became proof of concept for how a designer-strategist can ship functional products.
