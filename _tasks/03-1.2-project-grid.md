## Context
The asymmetric grid container that queries projects and lays them out.

## What to build
A grid component that fetches all projects from the content collection, sorts by order, and renders them in asymmetric layout with 1px dividers.

## Technical details

### Data fetching
```astro
---
import { getCollection } from 'astro:content';
import ProjectCard from './ProjectCard.astro';

const projects = (await getCollection('projects')).sort((a, b) => a.data.order - b.data.order);
---
```

### Grid layout
- Parent: display grid, grid-template-columns 1.4fr 1fr, grid-template-rows auto auto
- 1px divider technique: parent has `gap: 1px; background-color: var(--border)`. Children have `background-color: var(--bg-card)`.
- First child (order 1) gets class `.featured` with `grid-row: span 2`

### Responsive
- Tablet (<900px): grid-template-columns 1fr 1fr, first card no longer spans 2 rows
- Mobile (<768px): grid-template-columns 1fr, background none, gap 12px. Cards get explicit `border: 1px solid var(--border)`.

### Reveal
- Each card wrapped with `.reveal` and staggered transition-delay
