## Context
The HTML shell that wraps every page. Includes font preconnects, global CSS, meta tags, and the scroll observer script.

## What to build
An Astro layout component that provides the full HTML document structure.

## Technical details

### Structure
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>{title} — Lyla Huang</title>
  <meta name="description" content={description}>
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <!-- global.css imported via Astro -->
</head>
<body>
  <slot />
  <script> /* scroll observer */ </script>
</body>
</html>
```

### Props
- `title`: string (default: "Portfolio")
- `description`: string (default: "Lyla Huang — Strategist turned builder...")

### Scroll Observer Script
Inline `<script>` (not a module — runs immediately):
```javascript
const observer = new IntersectionObserver((entries) => {
  entries.forEach(e => {
    if (e.isIntersecting) e.target.classList.add('visible');
  });
}, { threshold: 0.08 });
document.querySelectorAll('.reveal').forEach(el => observer.observe(el));
```

### Import global.css
```astro
---
import '../styles/global.css';
---
```
