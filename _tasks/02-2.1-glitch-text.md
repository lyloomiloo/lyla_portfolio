# Task: Build GlitchText.astro — JS character swap island
Plan: `/_plans/02-hero.md` → Step 2.1
Spec: `/_specs/02-hero.md`

## Context
The signature interaction of the portfolio. Characters in the title periodically swap to random symbols then snap back — like a signal glitching.

## What to build
An Astro component with `client:load` that wraps text and runs two glitch effects: a one-time CSS clip-path glitch on load, and a periodic JS character swap.

## Technical details

### Component interface
```astro
---
interface Props {
  text: string;       // e.g. "LYLA\nHUANG."
  tag?: string;       // HTML tag to render, default "h1"
  class?: string;     // additional classes
}
---
```

Renders the text inside the specified tag. The `\n` in text becomes `<br>`. The last character (if it's a period) gets wrapped in `<span class="accent-dot">` for accent coloring.

### CSS glitch (one-time on load)
Apply the `glitchIn` animation (defined in global.css) to the element on mount:
```css
.glitch-enter {
  animation: glitchIn 0.35s ease-out forwards;
}
```

### JS character swap (periodic)
```javascript
const glitchChars = '*#@/\\][01{}<>_|~';

function glitch(el) {
  const original = el.dataset.original;
  const chars = original.split('');
  // Get indices of non-space, non-newline, non-tag characters
  const indices = [];
  chars.forEach((c, i) => {
    if (c !== ' ' && c !== '\n' && c !== '.' && c !== '<' && c !== '>') indices.push(i);
  });
  // Pick 1-3 random
  const count = Math.floor(Math.random() * 3) + 1;
  const shuffled = [...indices].sort(() => Math.random() - 0.5);
  const picks = shuffled.slice(0, count);
  // Replace
  picks.forEach(i => {
    chars[i] = glitchChars[Math.floor(Math.random() * glitchChars.length)];
  });
  el.textContent = chars.join('');
  // Restore after 80-120ms
  const delay = 80 + Math.random() * 40;
  setTimeout(() => { el.textContent = original; }, delay);
}

// Store original text
el.dataset.original = el.textContent;
// Run every 3-5 seconds
setInterval(() => glitch(el), 3000 + Math.random() * 2000);
```

### Important: handle innerHTML vs textContent
Since the title has a `<br>` and accent span, the glitch should operate on the raw text content only. Store the original `innerHTML` and restore it. The glitch swaps characters in a text-only copy displayed briefly, then restores the full HTML.

Alternative simpler approach: make the glitch target only the text nodes inside the element, not the HTML structure.
