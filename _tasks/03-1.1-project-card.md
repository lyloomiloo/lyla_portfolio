# Task: Build ProjectCard.astro
Plan: `/_plans/03-projects.md` → Step 1.1
Spec: `/_specs/03-projects.md`

## Context
Individual project card component used in the asymmetric grid.

## What to build
A card component that accepts project frontmatter and renders the card layout.

## Technical details

### Props (from content collection frontmatter)
- displayTitle: string (ALL CAPS)
- hook: string
- tags: string[]
- slug: string
- status: string
- year: string
- order: number
- thumbnail: string (optional)

### Structure
```html
<article class="project-card">
  <span class="project-index">0{order}</span>
  <div class="project-mockup">
    <!-- placeholder or thumbnail -->
  </div>
  <h3 class="project-name">{displayTitle}</h3>
  <p class="project-hook">{hook}</p>
  <div class="project-tags">
    {tags.map(t => <span>{t}</span>)}
  </div>
  <a href={`/projects/${slug}`} class="project-link">View Project →</a>
  <div class="project-meta">{status} · {year}</div>
</article>
```

### Styles
- Card: padding 1.8rem, background var(--bg-card), no explicit border (handled by grid parent)
- Index: var(--font-mono), 0.52rem, var(--text-dim), uppercase, letter-spacing 0.15em, margin-bottom 1.2rem
- Mockup: 16:10 aspect ratio, background var(--bg), border 1px solid var(--border), flex center, mono text placeholder, margin-bottom 1.2rem
- Name: var(--font-display), clamp(1.3rem, 2.5vw, 1.8rem), weight 700, var(--text), letter-spacing -0.03em, margin-bottom 0.4rem
- Hook: var(--font-body), 0.85rem, weight 300, var(--text-mid), line-height 1.65, margin-bottom 1rem
- Tags: flex wrap, gap 4px. Each span: var(--font-mono), 0.5rem, uppercase, letter-spacing 0.06em, padding 2px 7px, border 1px solid var(--border), color var(--text-dim)
- Link: var(--font-mono), 0.62rem, color var(--blue), uppercase, letter-spacing 0.1em. Hover: letter-spacing 0.16em (transition)
- Meta: var(--font-mono), 0.52rem, var(--text-dim), margin-top 1rem, padding-top 0.7rem, border-top 1px solid var(--border)
