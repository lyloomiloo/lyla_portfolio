## Context
Decorative retro UI fragments that float around the layout. They're the cyberfuturist texture — scattered debris from an old OS. Not interactive, purely visual.

## What to build
A reusable component that renders a set of absolutely-positioned decorative pixel elements based on a variant prop.

## Technical details

### Props
- `variant`: "hero" | "contact" | "section" — determines which set of artifacts to render

### Artifact types (all absolute positioned, pointer-events: none)
1. **Pixel square**: `<div>` with fixed width/height (6-12px), solid background color, `image-rendering: pixelated`
2. **× button**: `<div>` 13×13px, 1px solid border (#D0CCC4), containing "×" in var(--font-mono) 8px, color #CCC, bg var(--bg)
3. **Scrollbar fragment**: `<div>` 8px wide, variable height (40-60px), repeating-linear-gradient horizontal stripes
4. **Smiley**: `<span>` ":)" in var(--font-mono) 10px, color var(--text-dim), opacity 0.4
5. **Window corner**: Two `<div>`s forming an L-shape (two 1px borders meeting at a right angle)

### Variant configs
Each variant defines an array of artifacts with: type, top/right/bottom/left positioning (%), color, size, opacity.

Example for "hero" variant (~10 artifacts):
- Green square 10px, top 20%, right 20%
- Pink square 10px, top 32%, right 10%
- Grey square 8px, top 60%, right 30%
- Black square 6px, top 44%, right 5%
- × button, top 27%, right 34%
- × button, top 52%, right 16%
- Scrollbar 50px, top 36%, right 42%
- Green square 6px, top 72%, right 38%
- Pink square 6px, bottom 18%, right 7%
- Smiley, top 45%, right 25%

"contact" variant (~3 artifacts):
- Green square 10px, top -6px, right -6px
- Pink square 6px, bottom -5px, left -5px
- Grey square 4px, top 50%, right -10px

### Wrapper
The component renders a `<div>` with `position: relative` that wraps no content (artifacts are inside). The parent component should have `position: relative; overflow: hidden` to contain them.

On mobile (<768px): hide all artifacts (`display: none` on the wrapper).

- Opacity should vary (some at 0.3, some at 1.0) for depth
- Colors should mix green, pink, grey, and black — not all the same
