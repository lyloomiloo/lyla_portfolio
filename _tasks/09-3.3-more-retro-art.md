## Context
The robot pixel character is good — more of that. Add a second creature and retro window decorations (title bar with dots) for a richer pixel art presence.

## What to build
New RetroElements types: "pixel-creature" (a ghost/alien/cat), "window-bar" (retro OS title bar).

## Technical details

### pixel-creature
A 12x12 grid pixel art ghost/alien using the purple+yellow palette:
- 0=transparent, 1=purple (#7B2DFF), 2=yellow (#FFE600), 3=grey (#666), 4=white
- Design: cute ghost shape — rounded top, wavy bottom, two eyes, small mouth
- Animation: gentle float (translateY -3px to 0) over 2s
- Rendered in a dark grey (#2A2A2A) block with padding, same as robot strip
- Full-width dark strip container

### window-bar
A thin horizontal bar that looks like a retro OS window title:
```html
<div class="window-bar">
  <span class="window-dots">
    <span></span><span></span><span></span>
  </span>
  <span class="window-title">{title}</span>
  <span class="window-close">×</span>
</div>
```
- Bar: bg #333, height ~30px, flex items center, padding 0 12px
- Dots: 8px circles, colors purple/yellow/#666, gap 5px
- Title: mono 0.8rem, #888, centered
- Close: mono, #888, decorative only
- Props: `title` string
