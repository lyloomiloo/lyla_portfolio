# Task: Add retro buttons component
Plan: `/_plans/09-visual-overhaul.md` → Step 3.2
Spec: `/_specs/09-visual-overhaul.md`

## Context
Main CTAs ("View Project →", contact links) are currently plain text links. Retro chunky buttons make them more prominent and match the pixel art aesthetic.

## What to build
A RetroButton component and apply it to ProjectCard and Contact.

## Technical details

### RetroButton.astro
Props: `href`, `text`, `variant` ("purple" | "yellow" | "outline")

```html
<a href={href} class={`retro-btn retro-btn-${variant}`}>{text}</a>
```

### Styles
- Base: display inline-block, font-family var(--font-mono), font-size 0.85rem, text-transform uppercase, letter-spacing 0.1em, padding 8px 16px, cursor pointer, transition transform 0.1s
- Purple variant: bg var(--purple), color var(--yellow), border 2px solid var(--purple). Border-top/left: 1px solid rgba(255,255,255,0.2) (inset highlight). Border-bottom/right: 1px solid rgba(0,0,0,0.3) (inset shadow).
- Yellow variant: bg var(--yellow), color var(--dark), border 2px solid var(--yellow). Same inset effect.
- Outline variant: bg transparent, color var(--purple), border 2px solid var(--purple)
- Hover: transform translateY(-1px), box-shadow 0 2px 0 var(--dark)
- Active: transform translateY(1px), box-shadow none

### Apply in ProjectCard
Replace `<a class="project-link">View Project →</a>` with `<RetroButton href={...} text="View Project →" variant="purple" />`

### Apply in Contact
Wrap email and LinkedIn links in RetroButton with variant="outline"

## Files to create/modify
- `src/components/RetroButton.astro` — create
- `src/components/ProjectCard.astro` — replace link with button
- `src/components/Contact.astro` — replace links with buttons

## Acceptance criteria
- Buttons render with chunky retro style
- Inset shadow creates 3D pixel button feel
- Hover lifts button slightly
- Active presses button down
- Consistent across project cards and contact
