## Technical details

### New header structure (mirrors ProjectCard)
```
<span class="pd-index">0{order}</span>
<GlitchText text={displayTitle} tag="h1" class="pd-title" ... />

<div class="pd-meta">{status}</div>
```

### Styles (scaled up from card)
- pd-index: mono, 1rem, dim, uppercase, letter-spacing
- pd-title: display font, clamp(3rem, 8vw, 6rem), weight 700 (bigger than card's 1.8-3rem)
- pd-hook: body font, 1.3rem, weight 300, text-mid
- pd-tags: same bordered pills as card but slightly larger (0.85rem)
- pd-location: mono, 0.95rem, dim
- pd-meta: mono, 0.95rem, dim, border-top
