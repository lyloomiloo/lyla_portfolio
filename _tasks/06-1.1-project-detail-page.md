# Task: Build [slug].astro dynamic project pages
Plan: `/_plans/06-project-pages.md` → Steps 1.1 + 1.2
Spec: `/_specs/06-project-pages.md`

## Context
Each project needs a full detail page that renders from markdown content.

## What to build
A dynamic Astro page at `/projects/[slug]` that renders project writeups from the content collection.

## Technical details

### Route setup
```astro
---
import { getCollection } from 'astro:content';
import BaseLayout from '../../layouts/BaseLayout.astro';
import Nav from '../../components/Nav.astro';

export async function getStaticPaths() {
  const projects = await getCollection('projects');
  return projects.map(p => ({
    params: { slug: p.data.slug },
    props: { project: p },
  }));
}

const { project } = Astro.props;
const { Content } = await project.render();
---
```

### Page structure
1. Back link: `<a href="/#projects">← Back to projects</a>` — var(--font-mono), 0.65rem, var(--text-dim), hover var(--text)
2. Title: `<h1>{project.data.displayTitle}</h1>` — var(--font-display), large, ALL CAPS
3. Tagline: `<p>{project.data.tagline}</p>` — var(--font-body), 1rem, var(--text-mid)
4. Metadata block: role, duration, status, year — var(--font-mono), small, var(--text-dim)
5. Tools as tag pills (same style as index)
6. `<Content />` — rendered markdown body
7. Max content width: ~720px, centered with margin auto

### Prose styling
Style rendered markdown within a `.prose` wrapper:
- h2: var(--font-display), 1.4rem, weight 600, margin-top 2.5rem, margin-bottom 0.8rem
- h3: var(--font-display), 1.1rem, weight 600, margin-top 2rem, margin-bottom 0.6rem
- p: var(--font-body), 0.95rem, weight 300, line-height 1.75, margin-bottom 1.2rem, color var(--text-mid)
- ul/ol: margin-left 1.2rem, margin-bottom 1.2rem. li: margin-bottom 0.4rem
- strong: weight 500, color var(--text)
- em: style italic
