### Styles
- Section: min-height 100vh, display flex, flex-direction column, justify-content center, padding 8rem 2.5rem 4rem. Position relative, overflow hidden.
- Title: font-family var(--font-display), font-size clamp(4rem, 12vw, 10rem), weight 700, color var(--text), line-height 0.88, letter-spacing -0.05em
- `.accent-dot`: color var(--blue)
- Subtitle: font-family var(--font-mono), 0.72rem, color var(--text-dim), margin-top 1.5rem, uppercase, letter-spacing 0.15em

- Add `.reveal` classes with staggered transition-delays (0s, 0.08s, 0.16s)
