## Context
The dark terminal panel — the visual centerpiece of the about section. Black background with green monospace text showing a career timeline.

## What to build
A self-contained dark terminal component with header, scrollable log entries, and blinking cursor.

## Technical details

### Structure
```html
<div class="signal-log">
  <div class="signal-log-header">
    <span>signal_log.md</span>
    <span class="x-close">×</span>
  </div>
  <div class="signal-log-body">
    <!-- log entries -->
  </div>
</div>
```

### Styles
- Outer: background var(--bg-dark) (#0C0C0C), no border-radius
- Header: padding 10px 1.2rem, border-bottom 1px solid #222, font var(--font-mono) 0.55rem, color #555, uppercase, letter-spacing 0.1em, flex space-between. The × is color #444, decorative only.
- Body: padding 1.2rem, font var(--font-mono) 0.7rem, color var(--green), line-height 2, max-height 480px, overflow-y auto
- Custom scrollbar: width 6px, track #0C0C0C, thumb #1a3320

### Log entries (hardcoded content)
Each entry:
```html
<div class="log-entry">
  <span class="log-date">[2015]<span class="log-label">SIGNAL_START</span></span><br>
  <span class="log-text">Graduated NUS. Started writing ads.</span>
</div>
```

Color hierarchy:
- `.log-date`: color #1a6638, font-size 0.62rem, weight 500
- `.log-label`: color #00FF66 (bright green), weight 700, margin-left 0.3rem, font-size 0.58rem
- `.log-text`: color #4daa6a
- `.log-text .dim`: color #2a6642
- `.log-text .hl`: color #00FF66 (bright green), weight 500
- `.log-blockquote`: margin-top 0.5rem, padding-left 0.8rem, border-left 2px solid #1a4a2a, line-height 2.2

7 entries + 1 blockquote (exact text in CLAUDE.md about section). Final element is the blinking cursor: `<span class="cursor-blink"></span>` (6px wide, 0.85em tall, bg #00FF66, animation blink 1s step-end infinite).
