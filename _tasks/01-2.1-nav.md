## Context
Fixed navigation bar at the top of every page. Clean, minimal, matches the cyberfuturist aesthetic without being heavy.

## What to build
A fixed top navigation with name on left, section links on right, and a mobile hamburger menu.

## Technical details

### Desktop layout
- Fixed to top, full width
- Background: var(--bg) with border-bottom: 1px solid var(--border)
- Padding: 1rem 2.5rem
- Left: "Lyla Huang" in var(--font-display), 0.7rem, weight 700, uppercase, letter-spacing 0.12em, color var(--text)

- Links are anchor tags: `href="#projects"`, `href="#about"`, etc.
- z-index: 100

### Mobile (<768px)
- Hide link row, show hamburger icon (three horizontal lines, CSS-only)
- Hamburger toggles a full-screen overlay (background var(--bg), z-index 99)
- Overlay shows links vertically, larger font, centered
- Close button (×) in top right of overlay
- Use minimal inline JS: toggle a class on body or nav element

### Smooth scroll
- Already handled by `html { scroll-behavior: smooth }` in global.css
- Links use `href="#sectionId"`
