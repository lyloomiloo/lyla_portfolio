# Task: Apply GlitchText to project titles and section labels
Plan: `/_plans/07-design-refinements.md` → Step 3.2
Spec: `/_specs/07-design-refinements.md`

## Context
Spread the glitch effect across the page so it feels like a persistent ambient animation.

## What to build
Wrap additional text elements with GlitchText.

## Technical details

### Elements to wrap

| Element | Location | Intensity | Interval |
|---------|----------|-----------|----------|
| Hero title | Hero.astro | bold | 3000 |
| "(RE)ROUTE" | ProjectCard (order 1) | bold | 4000 |
| "OH SNAPP!" | ProjectCard (order 2) | bold | 4500 |
| "PLANMYTRIP" | ProjectCard (order 3) | bold | 5000 |
| "Projects" label | index.astro | subtle | 6000 |
| "About" label | index.astro | subtle | 7000 |
| "Capabilities" label | index.astro | subtle | 8000 |

### In ProjectCard.astro
The project name h3 should use GlitchText:
```astro
<GlitchText client:visible text={displayTitle} tag="h3" class="project-name" intensity="bold" interval={4000 + order * 500} />
```
Use `client:visible` (not `client:load`) so the glitch only starts when the card scrolls into view.

### In index.astro
Section labels:
```astro
<GlitchText client:visible text="Projects" tag="span" class="section-label-text" intensity="subtle" interval={6000} />
```

### Hero stays as-is
Hero already has GlitchText with `client:load` and `playEntrance={true}`.
